**Alien Box** — это современный мультимодальный механизм криминалистического аудиоанализа и генератор шаблонов DAW. Он подключается напрямую к глобальным музыкальным сетям для извлечения реальных метаданных из любого входного текста, анализирует частоту и динамические параметры коммерческих треков и генерирует индивидуальные стартовые шаблоны для основных DAW (Ableton, Logic Pro, Cubase, FL Studio, Pro Tools).

Создано с использованием точных технологий компанией **CHUS BZN** в **produktes-code**.

## 🚀 Особенности
- **Универсальный поиск.** Введите любую ссылку на YouTube, ссылку на Spotify или произвольный текст (например, «The Prodigy Poison»).
- **Жесткий судебно-медицинский анализ:** вычисляет целевые LUFS, конкретные кривые эквалайзера (кроссовер Kick/Bass) и время атаки/восстановления сжатия VCA.
- **Мультимодальный экспорт:** Создает настоящий ZIP-файл, содержащий предварительно маршрутизированный шаблон DAW.
- **Органический пользовательский интерфейс:** Гибкий, дышащий графический интерфейс с динамическим фоном из частиц.
- **Встроенное универсальное руководство в формате PDF (V14):** Руководство в формате PDF в высоком разрешении, симметрично переведенное на 7 языков, тщательно отформатированное и идеально масштабированное для печати формата DIN A5.

## 🛠 Установка

### 1. Настройка серверной части
Бэкэнд работает на легком сервере Python.
``` баш
# 1. Запустите сервер анализа
python3 Alienbox_server.py
```

### 2. Интерфейс
Просто откройте splash_screen_organic_v3_final/code.html в любом современном веб-браузере.

### 3. Создать руководство (V14)
Чтобы составить абсолютно симметричное универсальное руководство V14 в формате DIN A5:
``` баш
python3 build_universal_manual_v14.py
узел print_multilingual_v14_pdf.js
```

## 📦 Установщики сборок (macOS и Windows)
Чтобы упаковать Alien Box в собственное настольное приложение, мы предоставляем сценарии сборки для Electron/PyInstaller. Это подготавливает выпуск для нашего репозитория GitHub.

* **Mac OS (.dmg):** Запустите `./build_mac.sh`
* **Windows (.exe):** Запустите `build_win.bat`
